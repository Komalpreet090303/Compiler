This is the implementation of B-- Compiler for the final project for CS202:

Instructions to run the code-
1. Run the command 'yacc -y BMM_Parser.y' to comiple the yacc file.
2. Run the command 'lex BMM_Scanner.l' to compile the lex file.
3. Run the command 'gcc -g lex.yy.c y.tab.c' to run the code.
4. Run the command './a.out' to run the executable.
